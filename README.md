# Taqueria Cash Register

A simple Python script that simulates a taqueria (Mexican restaurant) cash register. Enter menu items, and it keeps a running total of the order.

## Features

- Supports a predefined menu of items with prices.
- Displays a running total after each item is entered.
- Ignores invalid items that are not on the menu.
- Exits gracefully when the user signals end-of-input (Ctrl+D on Linux/Mac, Ctrl+Z on Windows).
